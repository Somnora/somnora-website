## Deploy
- Static GitHub Pages site (remote `Somnora/somnora-website`, branch `main`, domain via `CNAME` = somnora.app). Push to `main` publishes; there is no build step and no CI.
- After editing `style.css` or `script.js`, bump the cache-bust query string in `index.html` (pattern `?v=YYYYMMDD<letter>`, e.g. `?v=20260707b`) or the CDN serves stale assets.

## Investor deck
- Do NOT hand-edit `investors/deck.html`. It is generated by `scripts/sync_investor_deck.sh`, which copies `~/Downloads/Somnora_Investor_Briefing.html`. Run the script, then commit + push.

## Gotchas
- The scroll-driven cloud parallax in `script.js` is document-anchored and historically glitchy — test scroll behavior on mobile and desktop after touching it.
- `PRICING` and `MODE_DETAILS` objects in `script.js` are the source of truth for pricing/mode copy shown on the page.
